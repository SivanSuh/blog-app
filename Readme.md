# MERN Stack Blog App

you can login or register and CRUD(Create, Read, Update and Delete) blog page

## Frontend Tech

- Next Js 13
- TypeScript
- Tailwind css
- Redux Toolkit
- Axios

## Backend Tech

- Node js
- Express js
- Mongo Db
- mongoose
- Bcrypt
